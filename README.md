# git-stat
Statistic generation for Git.

```
Usage: git-stat [options] <repository> [directories...]

  Options:

    -h, --help             output usage information
    -V, --version          output the version number
    -r, --recurse          Generate statistics recursively.
    -a, --author <author>  Get the commits by the specified author.
    -y, --year <year>      Get the commits for the specified year
```
